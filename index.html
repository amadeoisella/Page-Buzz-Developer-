Carrera


Conviertete en un Desarrollador Full-Stack a partir de las últimas tendencias de la industria, con la que vas a adquirir los conocimientos y capacidades técnicas para integrarte en el mercado laboral. Vamos a cubrir:



Python

Aprenderás a programar desde cero con python, sin tener ningún conocimiento sobre programación anterior.


MySQL

Aprenderás a usar este lenguaje de acceso a bases de datos que le permite al usuario ejecutar operaciones y administrar datos.


CSS

Con este lenguaje aprenderás a cambiar las fuentes, colores, estilos y animaciones de los elementos dentro de tu página web.


Javascript

JavaScript se utiliza como complemento de HTML y CSS para crear páginas webs, videojuegos, aplicaciones móviles.


Modalidad


Estudiá desde la comodidad de tu casa, aprendé a tu ritmo, mirá y pausá los videos las veces que quieras.

Modalidad

No hay un cronograma predefinido, cada módulo se paga por separado y se puede empezar cuando quieras, aprobando el módulo anteriormente.

Evaluación

Para aprobar cada módulo debés completar un desafío integrador que será evaluado por uno de los profes en github. No hay fecha límite para entregar el trabajo.

Certificado

Una vez aprobados los desafíos integradores de todos los módulos vas a recibir un certificado digital que podrás compartir con cualquier persona o entidad.




Quienes Somos

Somos un grupo de Desarroladores enfocados en las principales tecnologías requeridas en el mercado actual.


Misión Buzz

Queremos poder entregar una educación académica enfocada en capacitaciones de alto nivel y, conceptos desde lo más básico hasta lo más avanzado.
